🧠 Mô hình & Công nghệ

🤖

YOLOv12-Seg

Object Detection + Instance Segmentation, nhận diện 40 classes bao gồm phương tiện, đèn giao thông, vạch kẻ, biển báo

📊

ByteTrack

Multi-object tracking cho theo dõi phương tiện liên tục qua nhiều frame, hỗ trợ phân tích hành vi

⚡

FastAPI + WebSocket

Backend hiệu suất cao với streaming real-time, xử lý đồng thời nhiều client

🎯

Calibration System

Tự động hiệu chỉnh vùng vi phạm (vỉa hè, vạch dừng, làn đường) trong 5-10 giây đầu
